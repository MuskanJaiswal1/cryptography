FILE!SECURE

Secure File Storage Using Hybrid Cryptography

Objective

Build a trustworthy file protection platform for cloud workflows using chunk-based hybrid cryptography and intelligent strategy selection.

Core Method

  1. Upload any file type to the server.
  2. Split the file into fixed-size chunks.
  3. Encrypt chunks based on selected strategy.
  4. Protect key payload and provide key file to user.

Restore Flow

  1. Upload the key file from sender.
  2. Decrypt the stored algorithm keys.
  3. Decrypt chunks in the original order.
  4. Merge chunks and download the restored file.

ML-Based Adaptive Encryption

  • File analysis extracts size, type, entropy, and sensitivity hints.
  • Classifier predicts recommended security strategy.
  • Strategy options: STRONG BALANCED FAST
